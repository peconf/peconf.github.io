# PEConf Website

This repository contains the source for the PEConf website, published through GitHub Pages.

## About PEConf

PEConf is a community for product engineers.

The site is intentionally simple. Its job is to introduce PEConf clearly and briefly, not to behave like a company site, a recruiting page, or an event operations page.

The core message of the landing page is:

> We share what we learn while building products: how we work, what we struggle with, and how we grow.

## Deployment

- Repository: `peconf/peconf.github.io`
- GitHub Pages site: [https://peconf.github.io](https://peconf.github.io)
- Custom domain: [https://peconf.kr](https://peconf.kr)

The custom domain is defined in [`CNAME`](./CNAME).
